  REPO="${1:-anujj-ti/agentic-setup}"

Define a local function create_label that takes: name, color (without #), description. Runs:
  "$GH" label create "$name" --repo "$REPO" --color "$color" --description "$desc" --force 2>&1 >&2 || true
The --force flag updates existing labels (idempotent). The || true prevents set -e from killing the script if gh returns non-zero for an edge case.
