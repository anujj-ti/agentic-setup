Positional argument: PROPOSED_DOMAIN=$1 (required; exit 1 with JSON usage error if absent).

Implementation:
  OPENCLAW_CONFIG="$HOME/.openclaw/openclaw.json"
  [[ -f "$OPENCLAW_CONFIG" ]] || { echo '{"ok":false,"error":"openclaw.json not found at expected path"}'; exit 1 }

  AGENT_IDS=$(jq -r '.agents.list[].id' "$OPENCLAW_CONFIG" | sort)

  MATCH=$(echo "$AGENT_IDS" | grep -i "$PROPOSED_DOMAIN" || echo "")

  if [[ -n "$MATCH" ]]; then
    echo "{\"ok\":false,\"reason\":\"Agent already exists for domain '${PROPOSED_DOMAIN}': ${MATCH}\",\"action\":\"do_not_create\"}"
    exit 0
  fi

  echo "{\"ok\":true,\"action\":\"proceed_to_proposal\",\"checkedDomain\":\"${PROPOSED_DOMAIN}\",\"existingAgents\":$(echo "$AGENT_IDS" | jq -R -s 'split("\n") | map(select(length > 0))')}"
  exit 0

chmod +x the script. Note: both branches exit 0 — "domain exists" is not an error, it is a valid "do not proceed" signal. The Task Orchestrator reads the "ok" field to determine whether to proceed.
