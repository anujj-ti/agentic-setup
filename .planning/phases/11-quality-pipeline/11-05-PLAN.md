The script performs three registry searches, all best-effort with fallbacks per D-113:

  GH_COUNT=$(echo "$GH_STARRED" | jq 'length')
  Print to stdout: "- GitHub starred repos: ${GH_COUNT} potential matches found"
  If GH_COUNT > 0, also print the list via jq -r '.[] | "  * \(.name): \(.url)"'

SEARCH 2 — agentskills.io (follows 308 redirect per research):
  ENCODED=$(python3 -c "import urllib.parse,sys; print(urllib.parse.quote(sys.argv[1]))" "$PATTERN_DESCRIPTION" 2>/dev/null || echo "")
  AGENTSKILLS=$(curl -s --max-time 5 --location "https://agentskills.io/api/search?q=${ENCODED}" 2>/dev/null || echo "{}")
  Print to stdout: "- agentskills.io: searched (result: $(echo "$AGENTSKILLS" | jq -r 'if . == {} then "no results or unreachable" else "response received" end' 2>/dev/null || echo "unreachable"))"

SEARCH 3 — ClawHub (unreachable as of 2026-05-21 per research; log attempt):
  Print to stdout: "- ClawHub (clawhub.dev): unreachable as of 2026-05-21 research — no results logged"

Final output section:
  Print header: "## Registry Search Evidence"
  Print all three results
  Print: "- Conclusion: review results above — reuse/adapt if match found, author new if no match"

The script exits 0 in all cases (unreachable registry is not a failure). chmod +x the script.
