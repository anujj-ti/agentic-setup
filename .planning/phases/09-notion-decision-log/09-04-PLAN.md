## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-09-11 | Elevation of Privilege | --rollback-cmd execSync | accept | Command is constructed by Task Orchestrator from its own Beads task graph and decision log, not from raw user input. User Orchestrator validates the intent before delegating; Task Orchestrator constructs the rollback command. No raw user string reaches execSync. |
| T-09-12 | Denial of Service | execSync timeout on rollback command | mitigate | 30-second timeout on execSync; if it times out, error is captured and logged; revert-decision.js continues to mark the decision as pending_revert and informs User Orchestrator to retry the rollback manually. |
